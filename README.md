# Youtube-Popout-for-Chrome-Edge
Ermöglicht das Herauslösen von YouTube-Videos in ein rahmenloses  Popup-Fenster, ähnlich dem Opera-Browser-Feature. Der Nutzer kann  Größe und Startzeit frei wählen.

YT Pop-Out Player – Datenschutzerklärung

Diese Erweiterung erfasst, speichert oder überträgt keinerlei 
personenbezogene Daten. 

Die Erweiterung liest ausschließlich die Video-ID und den 
Abspielzeitpunkt des aktuell aktiven YouTube-Tabs aus, um das 
Video in einem Popup-Fenster zu öffnen. Diese Daten verlassen 
das Gerät des Nutzers nicht.

Es werden keine Cookies gesetzt, keine Analyse-Tools verwendet 
und keine Daten an Dritte weitergegeben.
